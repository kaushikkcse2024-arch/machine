Advisor (Cutting Data)

Machine Overview

Horizontal Milling Machine holds and drives the involute cutter through an arbor.

Dividing Head rotates the gear blank by a precise amount using the indexing crank.

Tailstock supports the blank between centers for alignment and rigidity.

Table Feed moves the work under the cutter to generate each tooth slot.

Calculation Menu

1. Gear Geometry

PCD = m x z
OD = m x (z + 2)
Addendum = 1.0 x m
Dedendum = 1.25 x m
Tooth Depth = 2.25 x m
  • m = module (mm)
  • z = number of teeth

2. Plain Indexing

Crank turns per tooth = 40 / z
Fractional turns use index plate holes
  • 40 = worm ratio of standard dividing head
  • z = required gear teeth
  • Example for z = 24: 40/24 = 1 + 10/15 turns

3. Milling (Material Removal Rate)

MRR = w x d x v
  • w = width of cut (mm)
  • d = depth of cut (mm)
  • v = feed rate (mm/min)
